## 🧠 Engineering Philosophy

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   "Every number is calculated, not generated.              │
│    AI interprets — it never invents."                      │
│                                                             │
│   1. Correctness first — wrong code that runs is           │
│      worse than no code.                                   │
│   2. Honest uncertainty — if I don't know, I say so.       │
│   3. Statistical grounding — compute in Python,            │
│      narrate with Claude.                                  │
│   4. Security by default — PDPA, prompt injection          │
│      defense, PII masking before any LLM call.            │
│   5. Verify before shipping — no "it works" without        │
│      actually checking the output.                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
